v0: Web app by MR to blur plates in pictures.

## Run Locally

To test the GitHub Pages-ready app in `v1`, run:

```bash
cd /home/ian/Workspace/plate-blur-webapp/v1
python3 -m http.server 8000
```

Then open:

```text
http://127.0.0.1:8000/
```
